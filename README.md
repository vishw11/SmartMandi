# SmartMandi - Farmer to Vendor Marketplace

SmartMandi is a full-stack web platform that connects **farmers directly with vendors**, helping eliminate middlemen and enabling fair pricing, transparent transactions, and efficient agricultural trade.

---

## Overview

Traditional agricultural markets often involve multiple intermediaries, reducing profits for farmers and increasing costs for buyers.

SmartMandi solves this problem by providing a digital marketplace where:

- Farmers can list their products directly  
- Vendors can browse and purchase fresh produce  
- Orders are managed online  
- Pricing becomes transparent  

---

## Key Features

### Farmer Module

- Farmer Registration / Login  
- Add Agricultural Products  
- Upload Product Images  
- Manage Inventory  
- Track Orders  
- Update Product Pricing  

### Vendor Module

- Vendor Registration / Login  
- Browse Available Products  
- Search / Filter Products  
- Add to Cart  
- Place Orders  
- View Order History  

### Admin Features *(if implemented)*

- User Management  
- Product Monitoring  
- Order Tracking
